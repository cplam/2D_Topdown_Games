# 🥷 Ninja Game – 2D Top‑Down Action

A classic top‑down action game where you step into the shoes of a nimble ninja. Explore a pixel‑art world, fight hostile creatures, and survive until every enemy is vanquished.


---

## ✨ Features

- **Retro Pixel Art** – hand‑crafted tiles, trees, houses, and terrain.
- **Smooth Canvas Rendering** – real‑time 2D graphics with no external libraries.
- **Combat System** – strike enemies while dodging their counter‑attacks.
- **Health Management** – keep an eye on your life bar; don't get hit too many times.
- **Win Condition** – clear the map of all monsters to claim victory.
- **Keyboard Controls** – intuitive movement and attack actions.

---

## 🎮 How to Play

| Action       | Key              |
|--------------|------------------|
| Move Up      | `W` or `↑`       |
| Move Down    | `S` or `↓`       |
| Move Left    | `A` or `←`       |
| Move Right   | `D` or `→`       |
| Attack       | `Space` or `J`   |

- Defeat all enemies on the map.
- Avoid enemy projectiles or melee hits.
- Your health is displayed at the top of the screen.

---

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)
- No server required – the game runs completely client‑side.

### Installation
1. Clone the repository or download the ZIP.
   ```bash
   git clone https://github.com/cplam/2D_Topdown_Games.git
   ```
2. Navigate to the `project_ninja_game` folder.
3. Open `index.html` in your browser.
4. Start playing immediately!
